## Hello, World!
```rust
fn main() {
    println!("Hello, world!");
}
```

`fn` means function. main function is the beginning of every Rust program.  
`println!` prints text to the console and its *!* indicate that itâ€™s a [macro](https://doc.rust-lang.org/book/first-edition/macros.html) instead of a function.

## Usages of println!

ðŸ’¯ These are the other usages of println! macro,

```rust
fn main() {
    println!("{}, {}!", "Hello", "world"); // Hello, world!
    println!("{0}, {1}!", "Hello", "world"); // Hello, world!
    println!("{greeting}, {name}!", greeting="Hello", name="world"); // Hello, world!

    println!("{:?}", [1,2,3]); // [1, 2, 3]
    println!("{:#?}", [1,2,3]);
    /*
        [
            1,
            2,
            3
        ]
    */

    // ðŸ”Ž format! macro is used to store the formatted STRING
    let x = format!("{}, {}!", "Hello", "world");
    println!("{}", x); // Hello, world!
}
```
